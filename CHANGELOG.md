## [0.1.0] - 2025-11-06

### Added
- Initial release of FamilyNotify
- Next.js 15 with App Router and TypeScript
- Supabase authentication and database
- Prisma ORM with complete schema
- Email notifications via Resend
- Web Push notifications (VAPID)
- SMS provider (Twilio stub)
- WhatsApp provider (Cloud API stub)
- Landing page with RTL support
- Onboarding flow
- User preferences management
- Announcements feed
- Events calendar
- Admin dashboard
- API routes for all operations
- Cron jobs for scheduled messages and reminders
- Row Level Security policies
- Dark mode support
- PWA manifest and service worker
- Comprehensive documentation
- Setup scripts and tooling

### Security
- Row Level Security (RLS) on all tables
- HTTPS only (enforced by Vercel)
- Security headers via middleware
- Input validation with Zod
- XSS protection
- CSRF protection

### Documentation
- README.md - Full documentation
- QUICKSTART.md - Quick start guide
- DEPLOYMENT.md - Deployment instructions
- RLS_POLICIES.md - Security policies
- API.md - API documentation
- CONTRIBUTING.md - Contribution guidelines
